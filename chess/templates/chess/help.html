Help page

Home

This is a high level overview of the data stored on the website.

On the left, you can see the names, ratings, grade, and coaches of the students top 10 students. I will add a checkbox to toggle the teachers into the list.

This also shows the Classes that are available for any student to be assigned to which will be reflected in the exported rating sheet!

On the right, you can see the number of games per week and the date of the match. This can be used to reference if next week's pairings have been made!

Search

This is where you can search for any specific keyword in the database.

The "Searching for" selection means where you want to search. Like if you want to search through all possible players, all possible games, or all players in a specific class.

I will also add a little pencil icon which would let you edit that specific result.

Manual Change

This hasn't been added yet, but this is where you can add a class or player manually. Or set up the database for the next year.

Input Results

This hasn't been added yet, but this is where you would search up the date and add the results. With any substitution that happened, you will have to edit that row manually. On submission, this will then update all the ratings based on the result of the game. Once you see a "success" message it is also safe to go to "Pair" and pair your next week of games!

Or you can click the "Download Ratings" which will download the current ratings of all players to your computer. Keep in mind, that this also makes a fild on the server too. So check the list of ratings to see if a rating sheet has already been made for this week!

Pair

This hasn't been added yet, but this is where you would have the computer do the pairings! If you have a specific match (such as a teacher vs student) in mind, you need to add those games BEFORE you hit "Pair!" After a success message, you can head over to "Download" tab.
