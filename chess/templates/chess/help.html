Home

On the home page, you can see the current ratings as stored in the database or pairings for any week in the database.

If you see an entry in the ratings sheet that is wrong, there is a pencil icon to the left. If you click this it will pop up an edit for that entry. Hit save and refresh the page and you should see the changes reflected on the home screen's view of pairing and ratings.

You can also use the search bar to find a keyword anywhere on the page such as a last name, class, or board number.

Manual Change

This is where you can add a player and/or class manually. Select what you want to add and a pop up will appear with the fields. Hit save and refresh the page and you should see the player. If you added a class, you can now add a player to that class by returning to the home page and hitting the pencil edit icon.

If you are trying to add a class but cannot find the teacher, you need to create the player first then you can assign a class to them!

If you are trying to add a player but cannot find the class, you need to create the class first then you can assign a player to that class!

You can also click the "Set Up Database for Next Year" which will ask you if you are sure. This will: add 1 to all active students' grade (removing those with grades higher than 5) and add 50 to everyone's ratings. Only do this when you are COMPLETELY finished with the current year's ratings and pairings.

As always, if you messed up something or find a bug, reach out to me for support. Contact info at the bottom of this page!

Input Results

Here is where you would go to add the results of a date's match. You first need to select the date of the match you are trying to change, then you should see a current pop up of all the games that were played on that date.

You can change the players on that board and/or the result of the game. If you need to have the board blank (as in no player on White and/or Black, scroll to the bottom to find 'N/A').

After you have added all the results (Please double check this!) hit the submit button.

After the submit button has been clicked, you should see a pop-up saying which should tell you which boards have been changed in the database and who's ratings have changed. After you see the ratings have been updated, you can now head over to "Pair" and pair your next set of games.

If you put in the result wrong and change it in this form, the ratings will be off. Do NOT do this, please contact me!

If you pair a student vs volunteer the student's rating will NOT change!

Under the "Download Ratings" section. This had two options:

  • Download a previous rating sheet already on the server
  • Create a new ratings sheet with information in the server, this will also save the file to the server for future reference

Pair

You first need to select the date of the match to be paired, then you should see a pop up appear. If you have a specific match (such as a teacher vs student) in mind, you need to add those games. You must put BOTH players on the board BEFORE you hit "Pair!". After a success message, you can download the pairings from that date.

If you have already submitted the pairing sheet and realized you have made a mistake, you need to use the input results' Update Game Results to fix this. Do NOT use the Make Pairing Sheet again!

If you are downloading a pairing sheet, it will ALWAYS download what is in the database at the moment you hit download. This does NOT save older versions of the pairing sheet for that date!
